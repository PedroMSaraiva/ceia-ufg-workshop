# 🧠 Workshop de NLP & MLOps — Do Modelo à Produção -TESTANDO NOVAMENTE2

📅 **23 a 25 de fevereiro de 2026**
📍 CEIA & ACKIT

## 📌 Visão Geral

Este workshop tem como objetivo capacitar participantes nos **fundamentos de Processamento de Linguagem Natural (NLP)** e nas **boas práticas de MLOps**, com foco na **implantação, orquestração, monitoramento e escalabilidade de modelos de IA em produção**.

Além do aprendizado técnico, o evento atua como um **chamado para novos talentos**, visando a formação de um time multidisciplinar nas áreas de **NLP, MLOps, DevOps e Visão Computacional**, contribuindo para a padronização e operação de soluções do **Centro de Excelência em Inteligência Artificial (CEIA)** e do **Centro de Competência em Tecnologias Imersivas (ACKIT)**.

---

## 🎯 Objetivos

* Introduzir conceitos essenciais de **NLP moderno**
* Ensinar práticas reais de **MLOps para produção**
* Demonstrar pipelines completos: **dados → treino → deploy → monitoramento**
* Capacitar participantes para atuar em **sistemas de IA escaláveis e confiáveis**
* Identificar talentos para integrar equipes de **pesquisa e engenharia aplicada**

---

## 👥 Público-Alvo

* Estudantes de Computação, IA, Engenharia ou áreas correlatas
* Pesquisadores interessados em IA aplicada
* Profissionais de Dados, Software ou DevOps
* Pessoas interessadas em atuar com **IA em produção**

---

## 🗓️ Estrutura do Evento

### 📍 Dia 1 — Fundamentos de NLP

* Introdução ao NLP moderno
* Pré-processamento de texto
* Representações vetoriais (TF-IDF, Word2Vec, embeddings)
* Transformers e LLMs
* Hands-on: treino de modelos NLP

### 📍 Dia 2 — Fundamentos de MLOps

* Versionamento de dados e modelos
* Pipelines automatizados (CI/CD para ML)
* Experiment tracking (MLflow, Weights & Biases)
* Deploy de modelos (FastAPI, Docker)
* Hands-on: pipeline de treino e deploy

### 📍 Dia 3 — Produção & Orquestração

* Orquestração (Airflow, Prefect, Kubernetes)
* Monitoramento de modelos e dados
* Observabilidade, drift e retraining
* Boas práticas para IA em produção
* Apresentação de projetos e **chamado para novos talentos**

## 🚀 Instalação

```bash
git clone https://github.com/seu-repo/workshop-nlp-mlops.git
cd workshop-nlp-mlops
pip install -r requirements.txt
```

Ou usando Docker:

```bash
docker build -t nlp-mlops-workshop .
docker run -p 8000:8000 nlp-mlops-workshop
```
